pub fn run(&mut self)

Go to user space with the context, and come back when a trap occurs.

On return, the context will be reset to the status before the trap. Trap reason and error code will be returned.

Example
use trapframe::{UserContext, GeneralRegs};

// init user space context
let mut context = UserContext {
    general: GeneralRegs {
        sp: 0x10000,
        ..Default::default()
    },
    sepc: 0x1000,
    ..Default::default()
};
// go to user
context.run();
// back from user
println!("back from user: {:#x?}", context);
